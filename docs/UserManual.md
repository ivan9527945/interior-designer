# RenderStudio 設計師使用手冊

## 前言

RenderStudio 是專為室內設計師打造的 AI 輔助渲染平台，讓您不必手動調整每一個燈光與材質參數，只需上傳設計圖、描述您想要的風格，系統便會自動在本機的 SketchUp／V-Ray 環境中完成高品質渲染，並將成果同步至雲端供您下載分享。本手冊將引導您從安裝 Agent 到完成第一張渲染圖的完整流程。

---

## 系統需求

| 項目 | 最低需求 |
|---|---|
| 作業系統 | macOS 14（Sonoma）或以上 |
| 瀏覽器 | Chrome 120+ 或 Safari 17+ |
| 3D 軟體 | SketchUp 2024 |
| 渲染引擎 | V-Ray 6 for SketchUp |
| 網路 | 穩定寬頻（上傳 DWG/SKP 需 5 Mbps 以上） |
| 儲存空間 | 本機至少 2 GB 可用空間（暫存渲染輸出） |

---

## 安裝 Agent

Agent 是運行在您電腦上的背景程式，負責接收渲染指令並驅動 V-Ray 執行。

**步驟 1：下載 Agent 安裝包**

前往 [https://app.renderstudio.io/download](https://app.renderstudio.io/download)，點選「下載 macOS Agent」，取得 `RenderStudio-Agent-macOS.zip`。

**步驟 2：解壓縮並移至應用程式資料夾**

將壓縮檔解壓縮後，把 `RenderStudioAgent` 資料夾移至 `~/Applications/` 或任意您偏好的位置。

**步驟 3：取得 API Token**

登入 RenderStudio 網頁平台 → 右上角「帳號設定」→「API Token」→ 點選「產生新 Token」並複製。

**步驟 4：設定 Agent**

在 `RenderStudioAgent` 資料夾中，找到 `config.yaml`，使用文字編輯器開啟並填入：

```yaml
api_base: https://api.renderstudio.io
api_token: 貼上您的-Token
sketchup_path: /Applications/SketchUp 2024/SketchUp.app
```

**步驟 5：啟動 Agent**

在終端機執行：

```bash
cd ~/Applications/RenderStudioAgent
python agent_main.py
```

Agent 啟動後，瀏覽器中 RenderStudio 右上角的圓點會變為綠色「Agent 已連線」，代表安裝成功。

## 第一次渲染完整流程

### 1. 上傳設計圖

登入後，點選左側「專案」→「新增專案」，輸入專案名稱後按「建立」。進入專案頁面，點選「上傳檔案」，選擇您的 `.dwg` 或 `.skp` 檔案，等待上傳完成。

### 2. 選擇風格

上傳完成後，系統會顯示風格選擇面板。您可以：

- **從預設風格選擇**：點選縮圖（北歐極簡、日式侘寂、現代奢華等）
- **輸入文字描述**：在文字欄位填寫，例如「溫暖木質調、大量自然光」
- **上傳參考圖**：拖曳一張您喜歡的參考照片，系統會分析其色調與氛圍（詳見下方「風格輸入三種方式」）

### 3. 設定畫質

在右側「渲染設定」欄位選擇輸出畫質：

- **快速預覽**：約 2–5 分鐘，適合確認構圖
- **標準**：約 10–20 分鐘，適合提案報告
- **高品質**：約 30–60 分鐘，適合最終交付

### 4. 送出並等待

確認設定後，點選「開始渲染」。您可以：

- 關閉瀏覽器，渲染在本機背景執行
- 完成後系統會以 Email 通知您
- 瀏覽器重新開啟後，可在「成果廊」看到完成的作品

### 5. 下載成果

進入「成果廊」，點選渲染圖縮圖開啟詳情頁，點選「下載原圖」取得高解析 PNG，或點選「分享連結」產生可供客戶瀏覽的連結。

## 風格輸入三種方式說明

**方式一：選擇預設風格標籤**

系統內建多種室內設計風格，點選縮圖即可套用，適合快速上手或不確定風格方向時使用。

**方式二：自由文字描述**

在「風格描述」欄位以自然語言輸入，例如：
- 「白色調極簡風，搭配溫暖木地板，大面積落地窗引入自然光」
- 「深色系工業風，裸露磚牆，黃銅燈具」

建議描述包含：整體色調、材質偏好、光線氛圍、家具風格。

**方式三：上傳參考圖片**

點選「上傳參考圖」，可上傳 JPG 或 PNG 圖片（建議解析度 1024×768 以上）。系統 AI 會分析圖片的配色、材質與氛圍，自動轉化為渲染風格指引。此方式最適合您有明確靈感圖片的情況。

## 常見問題

**Q1：上傳 DWG 後顯示「格式不支援」怎麼辦？**

請先在 AutoCAD 或 SketchUp 中將檔案另存為 AutoCAD 2018 版本以下的 `.dwg`，或直接匯出為 `.skp` 格式再上傳。

**Q2：Agent 顯示「未連線」但我確定已啟動程式？**

請確認 `config.yaml` 中的 `api_token` 是否正確，並確認電腦網路可連線至 `api.renderstudio.io`（部分公司 VPN 可能會封鎖此域名）。

**Q3：渲染已送出，但 30 分鐘後仍是「等待中」狀態？**

可能原因是 Agent 在渲染過程中意外關閉。請重新啟動 Agent，系統偵測到後會自動繼續未完成的任務。若仍無反應，請回到專案頁面點選「重新渲染」。

**Q4：成果圖的光線感覺不對，可以調整嗎？**

可以在風格描述中補充光線說明，例如「以暖色漫射光為主，避免強烈陰影」，然後重新送出渲染即可。每個專案可無限次重新渲染。

**Q5：我可以同時執行多個渲染任務嗎？**

單台電腦的 Agent 一次執行一個渲染任務。若需要並行渲染，可在多台電腦安裝 Agent 並登入同一帳號，系統會自動分配任務。

若以上說明仍無法解決您的問題，歡迎透過以下方式聯絡：

- **Email 技術支援**：support@renderstudio.io
- **回應時間**：工作日上午 9 時至下午 6 時，通常 2 小時內回覆
- **問題回報**：請附上「問題發生時間」與「渲染任務 ID」（可在詳情頁複製），有助於快速定位問題
